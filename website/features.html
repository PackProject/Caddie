(B)+(-)+(2): Button combination that resets the current shot before it has ended. If the Retry Shot Menu is enabled, a shot can also be reset after it has ended by selecting "Try Again".

Exception handler: A bunch of text that will show up if the game crashes, which might be Caddie's fault. If this happens to you, please send screenshots in the Discord!

Timer: Tracks the time that has passed since the last load into a hole. Pauses when the game is paused.

(+) Menu

The yellow ⬤ circle indicates a feature that has also been implemented in Frisbee Golf.

⬤ Hole: Selects a hole to play.

Repeat Hole: If set to Yes, the current hole will restart after it's completed. If set to No, completing a hole will transition to the next hole sequentially, as the game typically would.

Pin Type: Sets or randomly selects one of the pin locations in the current hole. For most holes, there are six of these, and Caddie can also randomize within two groups of them ("A pins" and "B pins").

  • A pins would normally occur in the game when the player's score is -5 or lower, and are meant to be more difficult.
  • B pins would normally occur in the game when the player's score is -4 or higher, and are meant to be more forgiving.
  • Pins in holes 19, 20, and 21 are not labeled as A or B because those holes are not part of any 9 or 18 hole course.
  • Pin selection is disabled on hole 18 because it only has one pin location. This is the only hole where this is the case.

⬤ Wind Selector: Specifies the direction and speed of the current wind. If the speed is set to Random, different speed randomization ranges can be selected that correspond with the randomization ranges that the game uses for different courses.

The wind selection tools allow for the creation of scenarios that would not normally exist in the game. For example, speeds greater than 20mph/10m/s are only included in Resort C, Classic C, and Special, but Caddie does not stop the user from selecting those speeds on any hole.

Show Replay Info: Renders a sphere centered around the hole that shows the area that will trigger a replay on the current shot. Replays become much more sensitive if the ball ever hits the pin or flag, which is reflected in the radius by expanding significantly and turning blue.

For putts on the green, Show Replay Info will not do anything if the current putt cannot replay from any angle. If it can, the putting guideline will have the same color-coding as the sphere: red if it won't replay from the current angle, and green if it will. Putts from off the green do not currently display any replay information.

Enable Retry Shot Menu: Prompts the user to either continue the game or retry the current shot after each stroke.

Apply and Restart: Applies changes to settings and restarts the hole. The settings in the (+) menu will not be reflected in the game until this option is selected.

Quit Game: Exits to the main menu.
